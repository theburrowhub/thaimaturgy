// FEATURES

🎲

D&D 5e Character System

Full character creation with races, classes, ability scores, inventory, conditions, and leveling.

🤖

AI Dungeon Master

Powered by OpenAI or Anthropic. The AI rolls dice, manages combat, and crafts immersive narratives.

💾

Save & Load

Persistent game sessions with auto-save. Continue your adventure anytime.

🎨

Retro TUI

CGA/EGA-inspired color palette with classic box-drawing characters. Pure terminal nostalgia.

🔊

Voice Narration

Optional text-to-speech narration using OpenAI TTS. Multiple voice options available.

🌍

Multi-Language

Full support for English and Spanish. The DM speaks your language.

// INSTALLATION

Go Install

go install github.com/theburrowhub/thaimaturgy/cmd/thaimaturgy@latest

From Source

git clone https://github.com/theburrowhub/thaimaturgy.git cd thaimaturgy make build ./bin/thaimaturgy

Set your API key:

# For OpenAI export OPENAI_API_KEY=sk-your-key # For Anthropic export ANTHROPIC_API_KEY=sk-ant-your-key

// COMMANDS

/roll 2d6+3 Roll dice
/save Save game
/load Load game
/status Character status
/inv Inventory
/quests Active quests
Ctrl+S Quick save
Ctrl+N Toggle voice

// SUPPORTED PROVIDERS

OpenAI

gpt-4o, gpt-4o-mini, gpt-4-turbo

Anthropic

claude-3-opus, claude-3-sonnet, claude-3-haiku
